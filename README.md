Use o comando venv para criar uma cópia virtual de toda a instalação do Python:

cd your-project
python3 -m venv env

Configure o shell para usar os caminhos venv para Python ativando o ambiente virtual:

source env/bin/activate

Agora é possível instalar pacotes sem afetar outros projetos ou a instalação global do Python:

pip install google-cloud-storage
pip install --upgrade google-cloud-speec

Se quiser parar de usar o ambiente virtual e voltar ao Python global, desative-o:

deactivate
